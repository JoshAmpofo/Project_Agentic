# Platform Playbook (formatting rules for generation)

Every generated post file contains, in this order:
1. `# <Angle title>`
2. `**Hook type:** <type>`
3. `**Post (copy-paste ready):**` followed by the body in a fenced block
4. `**Why this should travel:** <one line>`

Every post must trace back to the week's theme.

## LinkedIn
- Length: ~1300–1800 characters.
- Line-broken for skim-reading: short lines, blank lines between thoughts.
- First line is a standalone hook that lands before the ~210-char "see more" fold.
- One idea. Soft CTA (invite a comment or DM). 3–5 relevant hashtags at the end.

## X
- ≤280 characters per post.
- Single post by default. Use a thread (3–7 numbered posts) ONLY if the angle needs
  sequencing; render a thread as `1/`, `2/`, … each ≤280 chars.
- 0–2 hashtags. Punchy first 7 words. One clear payoff.

## Facebook
- 80–250 words, conversational and story-led.
- Relatable opener (not corporate). End on a genuine question to pull comments.
- Minimal/no hashtags.
