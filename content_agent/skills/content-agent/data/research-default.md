# Default Virality Playbook (evergreen fallback)

> Used when no `research-<niche>.md` cache exists. Run `/content-agent refresh <niche>`
> to replace this with niche-specific, dated research.

## Universal hook formulas (the first line decides everything)
- **Contrarian take:** "Everyone says X. They're wrong. Here's why."
- **Personal failure → lesson:** "I lost $X / wasted Y months doing Z. What I'd do now:"
- **Specific number list:** "7 things I wish I knew before <niche milestone>."
- **Open loop:** "Most people get <topic> backwards. The fix is simpler than you think:"
- **Status / identity:** "If you're a <audience>, stop doing <common mistake>."

## Format types that travel
contrarian take · listicle · personal story · how-to / playbook · data-drop /
surprising stat · myth-bust · behind-the-scenes · question/poll

## LinkedIn
- Skim-readable: short lines, whitespace, one idea per post.
- First 1–2 lines must hook before the "…see more" fold (~210 chars).
- Authority + vulnerability mix performs best; soft CTA (comment/DM), 3–5 hashtags.
- Best windows: Tue–Thu mornings.

## X (Twitter)
- Punch in the first 7 words; ≤280 chars per post.
- Threads only when the idea needs sequencing (3–7 posts), one payoff per post.
- 0–2 hashtags max; hooks > hashtags. Bold claims and concrete numbers travel.
- Best windows: midday and early evening.

## Facebook
- Conversational, story-led opener; relatable over corporate.
- 80–250 words; end on a question to pull comments (engagement bait that's genuine).
- Minimal hashtags. Native, personal tone outperforms link-drops.
- Best windows: midday.

## Cross-platform spacing principle
Quality over flooding: a feed that sees you 1–2x/day at the right time beats 5x of noise.
